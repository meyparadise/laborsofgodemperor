# Branding Colors — 12 Labors of GodEmperor

| Color | Hex | Use |
|-------|-----|-----|
| Pure White | `#FFFFFF` | Primary text, highlights |
| Gold | `#FFD700` | Accents, gradient top |
| Jade | `#00A86B` | Glow, borders, secondary accent |

## Thumbnail Text Style

- **"12 LABORS"**: Orbitron Black, gold gradient (`#FFD700` → `#B8860B`)
- **"GODEMPEROR"**: Exo Light, white (`#FFFFFF`), jade outline (`#00A86B`)
- **Drop shadow**: `#0D1117` at 80% opacity

## Usage

- **Thumbnails**: Use in GIMP text effects, overlays, and branding
- **Stream overlays**: Apply to OBS text sources, alerts, and banners
- **Social posts**: Consistent color hex codes across all platforms
- **Website/projects**: CSS variables or design tokens
